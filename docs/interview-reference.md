# 이차전지 실험 데이터 대시보드 — 면접 참고 문서

## 1. 서비스 개요

**이차전지(배터리) 실험 데이터를 조회·분석하는 웹 기반 대시보드**

- 실험 목록을 검색·필터링하여 원하는 데이터를 빠르게 찾을 수 있음
- 개별 실험의 충방전 데이터를 차트와 테이블로 시각화
- 여러 실험을 선택하여 성능을 비교 분석

---

## 2. 전체 페이지 흐름

```
/experiments (목록) → /experiments/:id (상세) → 개별 분석
                   ↘ /compare (비교) → 다중 실험 비교 분석
```

- `/experiments`: 전체 실험 목록. 필터링·검색 후 원하는 실험 선택
- `/experiments/:id`: 선택한 실험의 상세 데이터 확인 (차트 + 테이블)
- `/compare`: 여러 실험을 동시에 선택하여 성능 비교

---

## 3. 페이지별 역할 정리

### /experiments — 실험 목록

| 항목 | 내용 |
|---|---|
| **역할** | 전체 실험 데이터를 테이블 형태로 조회 |
| **사용자 행동** | 검색어 입력, 배터리 타입·온도 필터 적용, 컬럼 정렬, 실험 클릭하여 상세 이동 |

### /experiments/:id — 실험 상세

| 항목 | 내용 |
|---|---|
| **역할** | 특정 실험의 메타데이터, 사이클별 충방전 데이터를 차트·테이블로 표시 |
| **사용자 행동** | 용량 열화 추이 확인, 효율-온도 상관관계 확인, 사이클 데이터 수치 확인 |

### /compare — 비교 분석

| 항목 | 내용 |
|---|---|
| **역할** | 최대 8개 실험의 방전 용량·효율을 하나의 차트에 중첩하여 비교 |
| **사용자 행동** | 비교할 실험 체크박스 선택/해제, 차트에서 성능 차이 확인 |

---

## 4. 페이지 내부 구성 요소

### 필터 영역 (/experiments)

- **검색창**: 실험명·ID로 텍스트 검색
- **배터리 타입 필터**: NCM811, LFP, NCA 등 셀 화학 기준 필터링
- **온도 필터**: 실험 온도 조건 기준 필터링
- **필터 태그**: 적용 중인 필터를 시각적으로 표시, 개별 제거 가능

### 테이블

- **/experiments**: 실험 ID, 이름, 배터리 타입, 온도, 충전 조건, 사이클 수, 상태, 날짜 표시. 컬럼 클릭 시 정렬
- **/experiments/:id**: 사이클 번호, 충전 용량, 방전 용량, 효율, 온도, 전압을 수치로 표시 (상위 50 사이클)

### 차트 (상세 페이지)

- **Capacity vs Cycle**: 사이클에 따른 충전·방전 용량 변화
- **Efficiency & Temperature**: 효율과 온도의 사이클별 변화 (이중 Y축)

### 차트 (비교 페이지)

- **Discharge Capacity Comparison**: 선택된 실험들의 방전 용량 중첩 비교
- **Efficiency Comparison**: 선택된 실험들의 효율 중첩 비교

---

## 5. 차트(그래프) 설명

### 5-1. Capacity vs Cycle (용량 변화 차트)

| 항목 | 내용 |
|---|---|
| **X축** | 사이클 번호 (1, 2, 3, ...) |
| **Y축** | 용량 (mAh) |
| **파란선** | 충전 용량 — 각 사이클에서 배터리에 주입한 에너지 |
| **초록선** | 방전 용량 — 각 사이클에서 배터리가 출력한 에너지 |
| **인사이트** | 사이클이 진행될수록 용량이 감소하는 **열화(degradation) 추이** 확인. 충전-방전 간 차이가 클수록 에너지 손실이 큼 |

### 5-2. Efficiency & Temperature (효율·온도 차트)

| 항목 | 내용 |
|---|---|
| **X축** | 사이클 번호 |
| **좌측 Y축** | 효율 (%) = 방전 용량 / 충전 용량 × 100 |
| **우측 Y축** | 온도 (°C) |
| **파란 실선** | 충방전 효율 |
| **노란 점선** | 셀 온도 |
| **이중 Y축 이유** | 단위가 다른 두 지표(%, °C)를 하나의 차트에서 동시에 비교하기 위함 |
| **인사이트** | 온도 변화가 효율에 미치는 영향을 시각적으로 파악 |

### 라인 차트를 사용하는 이유

- 사이클 데이터는 **시간 순서가 있는 연속 데이터** → 라인 차트가 추세 파악에 가장 적합
- 수백~수천 사이클의 변화 패턴(열화 곡선)을 직관적으로 표현
- 여러 실험을 중첩할 때 선 간 차이를 쉽게 비교 가능

---

## 6. 비교 기능 설명

### 비교 방식

- 완료된 실험 목록에서 최대 8개를 체크박스로 선택
- 선택된 실험의 데이터를 **하나의 차트에 색상별로 중첩** 표시
- 선택/해제 시 차트가 즉시 갱신

### 비교를 통해 얻는 정보

- **열화 속도 차이**: 어떤 배터리 타입/조건이 수명이 긴지 비교
- **효율 안정성**: 사이클이 진행되어도 효율이 안정적인 실험 식별
- **조건별 성능 차이**: 온도, 충전 조건 등 변수에 따른 성능 차이 확인
- 예: NCM811 25°C vs LFP 45°C의 용량 유지율 비교
